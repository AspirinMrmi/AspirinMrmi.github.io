> HTTPS clone 和 SSH key 克隆区别？

HTTPS相对简单，复制url，执行git clone 即可。

使用SSH克隆则需要事先配置和添加SSH key

<!--more-->

> 查看自己是否已经创建了SSH key


```
cd ~/.ssh

ls
```
检查是否存在id_rsa.pub文件。


```
cat ~/.ssh/id_rsa.pub
```

查看密钥。

从ssh-rsa开始复制到结束添加到git。


> 创建SSH key

[here](https://help.github.com/en/articles/connecting-to-github-with-ssh)

> 在终端下展示树形目录结构


```
brew install tree
```

帮助信息


```
tree --help
```

目录遍历时使用 -L 参数指定遍历层级


```
tree -L n
```

将目录的结构树导出到md文件


```
tree -L 3 > x.md
```
