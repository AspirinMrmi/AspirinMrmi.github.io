环境搭建

安装NVM和Node.js

安装nvm

  • 通过curl： curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.2/install.sh | bash
  • 通过wget： wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.2/install.sh | bash

安装Node.js和NPM

nvm install v10.15.3

检查是否安装成功：

node -v, npm -v

安装webpack和webpack-cli

创建空目录和package.json

  • mkdir my-project
  • cd my-project
  • npm init -y

安装webpack和webpack-cli

  • npm install webpack webpack-cli --save-dev
  • 检查是否安装成功: ./node_modules/.bin/webpack -v
